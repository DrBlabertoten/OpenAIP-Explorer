# Chart Aggregator

> **Status: Phase 1 (Ideation)** - Actively researching user problems and defining the core value proposition.

## About The Project

**Chart Aggregator** is a comprehensive solution aimed at simplifying and centralizing free access to navigation charts for flight simulation enthusiasts. 

Currently, flight sim users face significant pain points when attempting to find, organize, and utilize accurate navigational charts without costly subscriptions. This project aspires to bridge that gap by aggregating necessary flight charts and integrating them with live network data (such as VATSIM) to provide a seamless flight tracking and navigation experience.
